File Structure and contents :
HIGH SCORES
The High Scores table files generated by the plugin commands are quite simple in fact. Here are their contents :

HEADER_STRING = "DBPExtendsPack_highscores"
Structure RECORD * 32
FINDEFICHIER_STRING = "_EOF"

Here is the RECORD structure mentionned below:
[Structure RECORD]
LONG = Score
LONG = level
STRING = Player name
[Structure End]
