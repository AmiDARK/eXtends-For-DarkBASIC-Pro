DLL Dependancies


Some DLL from the DarkBASIC Professional Extends Plugin pack need some other DLL from DarkBASIC Professional to be included to be sure the plugin commands work.
Here is the list of all Extends DLL dependancies.
To be sure that a DLL from Extends plugin pack work (left column), you must include one command of each DarkBASIC Professional DLL (middle column) because the plugin internally call functions from these DLLs. You must also include one command from each dll of the extends pack (right column) needed.
