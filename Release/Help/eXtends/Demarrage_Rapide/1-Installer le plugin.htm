How to install the plugin

if you don't have used the SETUP.EXE to install the plugin, here are the steps to reproduce to install it manually :

1. copy the drawers called : COMPILER, EDITOR, HELP et SAMPLES in the drawer where your DarkBASIC Professional copy is installed.
(default : c:\program files\darkbasic professional\)

WARNING : to work correctly, the plugin need these components to be installed ::
- DarkBASIC Professional (Online version or CDrom version)
- DarkBASIC Professional upgrade 6.0 final or higher
- Direct X9.0c ( February 2006 version )
