How to uninstall the plugin

if you used SETUP.EXE file to install the plugin, you can then uninstall it. To do this, follow this procedure :

1. Clic on button "START" from your desktop and then on "PARAMETERS"




2. Clic on "CONTROL PANEL".

3. Clic on 'Add/Remove softwares', find the line called 'DarkBasic Professionnel Extends Pack'.

3. Clic on the line : 'DarkBasic Professionnel Extends Pack' and on "UNINSTALL" button to proceed..

4. Follow the instructions to uninstall the plugin.
