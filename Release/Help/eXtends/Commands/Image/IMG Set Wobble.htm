IMG SET
This function will modify the default Wobble FX parameters. The Amplitude parameter define the maximum image distortion in pixels, speed define the distortion speed (the step used to calculate next cosinus value) and Step is used to define the shift in cosinus angle for the next image regarding to the last one.

SYNTAX
IMG Set Wobble Amplitude, Speed, Step
