IMG MAKE IMAGE
That command will create a new image by grabbing it from the current bitmap. You must enter the top,left coordinates and the bottom,right one that represent the box that border the image you want to create. It work in the same way than GET IMAGE from default DarkBASIC Professional command set.

SYNTAX
ImageNumber = IMG Make Image( Left, Top, Right, Bottom )
ImageNumber = IMG Make Image( Left, Top, Right, Bottom, MipMapMode )
