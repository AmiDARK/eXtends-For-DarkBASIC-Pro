IMG MOSAIC
This function will create a new image that will be modified from the original one with a Mosaics effects. You can specify target image if the image exist.

SYNTAX
TargetImage = IMG Mosaic( SourceImage, MosaicFactor, RandomMode )
TargetImage = IMG Mosaic( SourceImage, MosaicFactor, RandomMode, TargetImage )
