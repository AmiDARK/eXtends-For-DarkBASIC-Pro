MEMBLOCKS 


MEMBLOCKS COMMANDS :

MBC Initialized : MBC Initialized
MBC Clear All Memblocks : MBC Clear All Memblocks
MBC Exist Memblock Count : MBC Exist Memblock Count
MBC Delete Memblock : MBC Delete Memblock MemblockNumber
MBC Clone Memblock : MBC Clone Memblock SourceMemblock
MBC Make Memblock : MBC Make Memblock BytesSize
MBC Make Memblock From Bitmap : MBC Make Memblock From Bitmap BitmapNumber
MBC Make Memblock From Image : MBC Make Memblock From Image ImageNumber
MBC Make Memblock From Mesh : MBC Make Memblock From Mesh MeshNumber
MBC Make Memblock From Sound : MBC Make Memblock From Sound SoundNumber
MBC Make Memblock Image : MBC Make Memblock Image Width, Height, Depth
MBC Stretch Image : MBC Stretch Image [Source, Target] or [Source, Width, Height]

2D MEMBLOCKS COMMANDS :

MBC Plot : MBC Plot MemblockNumber, X, Y, RGBColor
MBC Circle : MBC Circle MemblockNumber, X, Y, Diameter, RGBColor, FillMode
MBC Cube : MBC Cube MemblockNumber, X, Y, Size, RGBColor, FillMode
MBC Box : MBC Box MemblockNumber, Left, Top, Right, Bottom, RGBColor, FillMode
MBC Get Pixel : MBC Get Pixel( MemblockNumber, X, Y )
