BMP LOAD BITMAP
That command will create a new bitmap from an image stored in a file (on disk). It handle all image files formats that DarkBASIC Professional handle. It work in the same way than usual LOAD BITMAP command from default DarkBASIC Professional command set.

SYNTAX
BitmapNumber = BMP Load Bitmap( Filename$ )
