BMP MAKE BITMAP FROM MEMBLOCK
That command will create a new bitmap from an image stored in a memblock. It work in the same way than usual MAKE BITMAP FROM MEMBLOCK command from default DarkBASIC Professional command set.

SYNTAX
BitmapNumber = BMP Make Bitmap From Memblock( MemblockNumber )
