DSK GET HIGHSCORE NAME$
That command will return the name of the player that did make the highscore at position 'HighScorePosition' in the highscore table. Check also commands : DSK GET HIGHSCORE LEVEL, DSK GET HIGHSCORE for other highscores informations.

SYNTAX
Player$ = DSK Get HighScore Name$( HighScorePosition )
