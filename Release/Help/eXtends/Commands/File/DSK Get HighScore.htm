DSK GET HIGHSCORE
That command will return the score reached by the highscore at position 'HighScorePosition' in the highscore table. Check also commands : DSK GET HIGHSCORE NAME$, DSK GET HIGHSCORE LEVEL for other highscores informations.

SYNTAX
Score = DSK Get HighScore( position dans la liste )
