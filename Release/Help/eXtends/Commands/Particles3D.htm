3D PARTICLES 


3D PARTICLES COMMANDS :

P3D Initialized : P3D Initialized
P3D Add Particle : P3D Add Particle Count, Texture, ParticleSize
P3D Position Emitter : P3D Position Emitter ParticleID, XPos, YPos, ZPos
P3D Set Particle Area : P3D Set Particle Area ParticleID, XSize, YSize, ZSize
P3D Set Primitive Mode : P3D Set Primitive Mode ParticleID, XAdd, YAdd, ZAdd
P3D Set As Flames : P3D Set As Flames Particle ParticleID
P3D Set As Smoke : P3D Set As Smoke Particle ParticleID
P3D Set As Rain : P3D Set As Rain Particle ParticleID
P3D Set As Snow : P3D Set As Snow Particle ParticleID
P3D Set As Primitive Particle : P3D Set As Primitive Particle ParticleID
P3D Update Particles : P3D Update Particles
P3D Delete Particle : P3D Delete Particle ParticleID
P3D Get Particle Exist : P3D Get Particle Exist ParticleID
P3D Clear Particles : P3D Clear Particles
P3D Hide Particle : P3D Hide Particle ParticleID
P3D Show Particle : P3D Show Particle ParticleID
