MBC CLEAR ALL MEMBLOCKS
That command will delete all memblocks that were previously created by the plugin sprites commands. (MBC MAKE MEMBLOCK, MBC MAKE MEMBLOCK FROM BITMAP, MBC MAKE MEMBLOCK FROM IMAGE, MBC MAKE MEMBLOCK FROM MESH, MBC MAKE MEMBLOCK FROM SOUND, MBC CLONE MEMBLOCK )

SYNTAX
MBC CLEAR ALL MEMBLOCKS
