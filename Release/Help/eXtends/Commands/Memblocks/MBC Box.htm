MBC BOX
That command will draw a box in an image stored in a memblock. Top,Left,bottom,Right are the limits of the box and flag Fillmode will fill entirely box if =1 or draw only the border if =0.

SYNTAX
MBC Box Memblock Number, Left, Top, Right, Bottom, Pixel Color, FillMode
