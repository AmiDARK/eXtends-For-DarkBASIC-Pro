MBC MAKE MEMBLOCK FROM SOUND
That command will create a new memblock from an image. The specified values must be entered using integer and resources must exist or the command may fail. Memblock will keep the image informations such as BitsPerSecond(DWord), Frequency(DWord), Duration(DWord), Sound Data(s)(Bytes). Memblock number will be defined by the function itself and returned to user as an integer value.

SYNTAX
MemblocNumber = MAKE MEMBLOCK FROM SOUND( SoundNumber )
