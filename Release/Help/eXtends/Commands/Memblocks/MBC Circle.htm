MBC CIRCLE
That command will draw a circle in an image stored in a memblock. X, Y are coordinate of the centre of the circle, Diam is the circle diameter and flag Fillmode will fill entirely circle if =1 or draw only the border if =0.

SYNTAX
MBC Circle Memblock Number, X, Y, Diam, Pixel Color, FillMode
