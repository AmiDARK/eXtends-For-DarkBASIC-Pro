MBC EXIST MEMBLOCK COUNT
That command will return the amount of memblocks created with the plugin memblock commands. (MBC MAKE MEMBLOCK, MBC MAKE MEMBLOCK FROM BITMAP, MBC MAKE MEMBLOCK FROM IMAGE, MBC MAKE MEMBLOCK FROM MESH, MBC MAKE MEMBLOCK FROM SOUND, MBC CLONE MEMBLOCK )

SYNTAX
Null = MBC Exist Memblock Count()
