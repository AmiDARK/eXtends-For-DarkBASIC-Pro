MBC STRETCH IMAGE
That command will stretch a memblock image. It can work using 2 ways defined below. In the first case, it will stretch the SourceMemblock image to fit the TargetMemblock image height and width. In the second case, it will create a new image that will have width and height size defined.

SYNTAX
MBC STRETCH IMAGE SourceMemblock, TargetMemblock
TargetMemblock = MBC STRETCH IMAGE SourceMemblock, Width, Height
