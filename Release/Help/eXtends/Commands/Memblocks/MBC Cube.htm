MBC CUBE
That command will draw a cube in an image stored in a memblock. X, Y are coordinate of the of the top left dot of the cube, size is the width and height size and flag Fillmode will fill entirely cube if =1 or draw only the border if =0.

SYNTAX
MBC Cube Memblock Number, X, Y, Size, Pixel Color, FillMode
