E3D LOAD EFFECT
This command will load an effect from a file. It work in the same way than default DarkBASIC Professional's LOAD EFFECT command excepted that, the number of the loaded effect will be choosen and returned to user by the command.

SYNTAX
Effect Number =E3D Load Object( Filename$, TextureMode )
