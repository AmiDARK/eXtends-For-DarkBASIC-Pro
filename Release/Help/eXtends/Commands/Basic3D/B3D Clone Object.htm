B3D CLONE OBJECT
This command will create a clone of an existing 3D object. It work in the same way than default DarkBASIC Professional's CLONE OBJECT command excepted that, the number of the created object will be choosen and returned to user by the command.

SYNTAX
ObjectNumber = B3D Clone Object( SourceObject )
