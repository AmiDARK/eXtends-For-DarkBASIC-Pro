B3D OBJECT TO BILLBOARD
This command will add object to the "billboard list". This list is used to make object always face the camera. It is used to create some sorts of particles. All billboards objects are update with the command B3D UPDATE BILLBOARDS that must be called just before the SYNC command.

SYNTAX
B3D Object To Billboard ObjectNumber
