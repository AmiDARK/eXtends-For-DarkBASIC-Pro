B3D MAKE OBJECT
This command will create a newobject from a mesh. It work in the same way than default DarkBASIC Professional's MAKE OBJECT command excepted that, the number of the created object will be choosen and returned to user by the command.

SYNTAX
ObjectNumber=B3D Make Object( MeshNumber, TextureNumber )
