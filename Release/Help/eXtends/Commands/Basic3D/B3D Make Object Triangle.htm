B3D MAKE OBJECT TRIANGLE
Cette commande permet à l'utilisateur de créer un objet de type "Triangle" en entrant les coordonnées X,Y et Z des 3 points composant le triangle. Le numéro de l'objet crée sera défini par la commande et renvoyé à l'utilisateur sous forme d'entier.

SYNTAXE
Nombre Entier Renvoyé=B3D Make Object Triangle( X1, Y1, Z1, X2, Y2, Z2, X3, Y3, Z3 )
