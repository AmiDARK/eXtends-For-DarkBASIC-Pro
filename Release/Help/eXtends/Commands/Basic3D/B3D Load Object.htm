B3D LOAD OBJECT
This command will load a 3D Object from a file. It work in the same way than default DarkBASIC Professional's LOAD OBJECT command excepted that, the number of the created object will be choosen and returned to user by the command. Flag is optional and work the same way LOAD OBJECT Filename$, Flag work.

SYNTAX
ObjectNumber =B3D Load Object( Filename$(, flag) )
