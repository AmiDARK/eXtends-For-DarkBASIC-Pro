B3D MAKE OBJECT CONE
This command will create a cone object. It work in the same way than default DarkBASIC Professional's MAKE OBJECT CONE command excepted that, the number of the created object will be choosen and returned to user by the command.

SYNTAX
ObjectNumber = B3D Make Object Cone( Size )
