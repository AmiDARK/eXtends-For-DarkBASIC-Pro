XGUI SET CHAT SCANCODE
 This command will define which key will start a chat text capture when pressed. You must chosse the ASCII value. It's the number your get when you ask for Scancode(). When that key will be pressed, the player will enter chat text in the gadget defined by XGUI SET CHAT GADGET.

SYNTAX
XGui Set Chat Scancode ASCIIVALUE
