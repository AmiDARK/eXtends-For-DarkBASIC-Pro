XGUI WINDOW PROPERTIES
This command define windows visual properties. If parameter 'borders' is set to 1, the window border will be displayed on screen. Otherwise, they will be hidden. if parameter 'title' is set to 1, the window title will be displayed (only if borders parameter is also set to 1) otherwise it will not be displayed. If 'dragging' parameter is set to 1, the window can be moved by clicking in the title bar like for generic windows.

SYNTAX
XGui Window Properties WindowNumber, Borders, Title, Dragging
