XGUI ATTACH WINDOW
 This command will attach a window to another one. The ChildWindow will then be dependant on Masterwindow. The parameter PositionID will define in which position the ChildWindow will be attached to its Master one.(1=Top, 2=Bottom, 3=Left, 4=Right)

SYNTAX
XGui Attach Window MasterWindow, ChildWindow, PositionID
