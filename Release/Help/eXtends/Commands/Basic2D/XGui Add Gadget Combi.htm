XGUI ADD GADGET COMBI
 This command will create a new gadget in a XGUI Window. The Gadget Combi will create a gadget that'll combinate text and graphic output.

SYNTAXE
XGui Add Gadget Combi WindowsNumber, GadgetNumber, X, Y, Width, Height, ImageNumber, String$
