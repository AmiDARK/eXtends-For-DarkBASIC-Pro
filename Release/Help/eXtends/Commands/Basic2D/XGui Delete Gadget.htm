XGUI DELETE GADGET
 This command will delete a gadget that was previously created using one of the XGUI Gadget commands. (XGUI ADD GADGET, XGUI ADD GADGET COMBI, XGUI ADD GADGET TEXT)

SYNTAX
XGui Delete Gadget( GadgetNumber )
