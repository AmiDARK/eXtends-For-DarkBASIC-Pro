XFONT SET CHAT GADGET
 This command will define which gadget will be used for chat system. It's a command created for multiplayer games and role playing games too. Player will start enter text in chat gadget when pressing the key defined by the command XGUI SET CHAT SCANCODE.

SYNTAX
XFont Set Chat Gadget GadgetNumber
