XFONT AUTO RETURN MODE
 This command will define the way the XFONT PRINT command will react when reaching an end of a line. If the parameter flag is set to 1, when the XFONT PRINT command reach the right border of the screen, it will automatically jump XFONT SET CURSOR to the beginning of the next line. If the flag is set to 0, the XFONT PRINT command will not print text that may appear out of the bitmap.

SYNTAX
XFont Auto Return Mode Flag
