B3D MAKE MESH FROM MEMBLOCK
This command will create a new mesh from a memblock. It work in the same way than default DarkBASIC Professional's MAKE MESH FROM MEMBLOCK command excepted that, the number of the created mesh will be choosen and returned to user by the command.

SYNTAX
MeshNumber = M3D Make Mesh From Memblock( SourceMemblock )
