EXT Initialized
That command will give 1 if the eXtends plugin is correctly initialized.
This command MUST be insered at the beginning of each project that will use the eXtends plugin. if you don't put that function at the beginning of your project, you'll get an error message saying that : "DBProExtends\GetDBProBase init failed"

SYNTAX
EXTInitialized = EXT Initialized()
