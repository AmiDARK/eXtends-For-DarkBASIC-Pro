L3D SET VIRTUAL LIGHT COLOR
This command will change the color of a virtual light. Red, Green and Blue are the three color components with a range from -255 up to 255. If the Virtual light is displayed in the 3D Scene, its color will change with the next L3D DISPLAY LIGHTS and SYNC calls.

SYNTAX
L3D Set Virtual Light VLightNumber, Red, Green, Blue
