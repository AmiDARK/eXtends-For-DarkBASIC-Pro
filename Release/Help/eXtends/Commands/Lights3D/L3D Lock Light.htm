L3D Lock Light
 This command will lock a true DarkBASIC Professional Light. There are7 available lights (1-7) and Light 0 That can't be used because the RTSkySystem use it. If you lock a true light, it will not be used to display Virtual lights. You can then use it on your own.

SYNTAX
L3D Lock Light LightNumber
