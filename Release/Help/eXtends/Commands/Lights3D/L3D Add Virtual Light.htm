L3D Add Virtual Light
This command will create a new Virtual Light. These lights are not directly visible in the 3D scne. In fact, they're added in a list and, the command L3D Display Lights will only display the one that are the nearest of the control camera defined by the command L3D SET CONTROL CAMERA. X,Y,Z parameters are the position of the Virtual Light in the 3D scene. Range define the light range. Red, Green, Blue are the 8bits colors components.

SYNTAX
VLightNumber = L3D Add Virtual Light( X, Y, Z, Range, Red, Green, Blue, Flag )
